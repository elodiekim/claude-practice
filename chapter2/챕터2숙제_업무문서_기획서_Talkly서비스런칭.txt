[기획서] AI 기반 직장인 영어 학습 서비스 'Talkly' 신규 런칭 기획안

작성자: 프로덕트팀 강하늘
작성일: 2026년 4월 25일
문서 버전: v1.2 (검토용)

──────────────────────────────────────────────

1. 기획 배경

최근 3년간 국내 성인 영어 학습 시장은 연평균 12% 성장하며 약 2조 5천억원 규모에 도달했다.
특히 30~40대 직장인을 대상으로 한 '실무 회화' 영역이 가장 빠르게 성장 중이며,
ChatGPT 등 LLM 기반 서비스가 등장한 이후 'AI 튜터' 형태의 학습 수요가 급증하고 있다.
그러나 기존 서비스(스픽, 링글 등)는 대부분 일반 회화에 초점이 맞춰져 있어,
직장인이 실제 업무에서 마주하는 상황(영어 회의, 이메일, 협상 등)에 특화된 솔루션은 부족한 상황이다.

2. 서비스 개요

서비스명: Talkly (가칭)
한 줄 정의: "직장인의 진짜 업무 영어를 24시간 연습하는 AI 튜터"
핵심 가치: 실제 업무 시나리오 + 개인 맞춤 피드백 + 무제한 연습

3. 타겟 고객

- 1차 타겟: 외국계/글로벌 기업 재직 30~40대 직장인 (영어 사용 빈도 주 1회 이상)
- 2차 타겟: 해외 진출 준비 중인 국내 기업 재직자
- 페르소나 1: 35세 IT 회사 PM, 분기별 해외 클라이언트 미팅, 회의 영어가 부담
- 페르소나 2: 41세 무역회사 과장, 매일 영어 이메일 작성, 표현이 제한적

4. 핵심 기능

(1) 시나리오 기반 회화 연습
- 회의 발표, 1:1 미팅, 협상, 면접, 컨퍼런스 콜 등 50+ 시나리오 제공
- 음성 입력 → AI 응답 → 실시간 피드백
- 사용자 발화 분석: 발음, 문법, 자연스러움 3개 축 점수화

(2) 업무 이메일 코칭
- 사용자가 작성한 이메일 → AI가 톤/문법/비즈니스 매너 관점에서 첨삭
- 상황별 템플릿 200종 제공 (사과, 거절, 리마인드, 협상 등)

(3) 개인 맞춤 학습 경로
- 첫 진단 테스트로 레벨 측정 (CEFR 기준)
- 약점 영역 우선 학습 → 4주 단위 리포트 자동 생성

(4) 챌린지 & 커뮤니티
- 매일 5분 챌린지 (출근길 활용)
- 직무별 그룹 (마케터, 개발자, 영업 등)

5. 기술 스택 (예정)

- LLM: Claude / GPT-4 계열 API 활용 (이중 구조)
- 음성 인식: Whisper 기반 자체 파인튜닝
- 음성 합성: ElevenLabs API
- 백엔드: Python(FastAPI), PostgreSQL, Redis
- 프론트엔드: React Native (iOS/Android 동시 지원)
- 인프라: AWS (Seoul Region)

6. 비즈니스 모델

- 무료 플랜: 일 1회 연습, 기본 시나리오 5종
- 프리미엄 월 19,900원: 무제한 연습, 전체 시나리오, 이메일 코칭
- 프로 월 39,900원: 1:1 인간 튜터 월 2회 + 프리미엄 전 기능
- B2B 기업 라이선스: 100인 기준 월 1,200만원 (별도 협의)

7. 일정 (Phase별)

- Phase 1 (2026.05~07): MVP 개발, 핵심 시나리오 20종 + 음성 회화 기능
- Phase 2 (2026.08~09): 클로즈드 베타 (300명), 피드백 반영
- Phase 3 (2026.10): 정식 출시 (iOS, Android, Web)
- Phase 4 (2026.11~12): B2B 영업 본격화, 기업 고객 10곳 확보 목표

8. 예산 (Phase 1~3)

- 개발 인력 (PM 1, 개발자 4, 디자이너 1): 월 5,500만원 × 6개월 = 3.3억
- API 사용료 및 인프라: 월 800만원 × 6개월 = 4,800만
- 마케팅 (베타 모집 + 출시): 1.5억
- 기타 (법무, 회계, 사무환경): 3,000만
- 총합: 약 5.6억

9. KPI

- 출시 6개월 후 MAU 5만명
- 유료 전환율 8% 이상
- 월 매출 4억원 달성 (출시 6개월 시점)
- B2B 계약 10건, 누적 1,000인 라이선스

10. 주요 리스크

- (시장) 대형 사업자(스픽, 듀오링고)의 직장인 특화 기능 출시 가능성
- (기술) AI 음성 응답 지연 시 사용자 경험 저하 → 캐싱 및 사전 응답 전략 필요
- (운영) 클로즈드 베타 단계에서 시나리오 다양성 부족 시 이탈률 증가 우려
- (재무) 광고비 효율 악화 시 CAC가 LTV를 초과할 가능성 → 초기 오가닉 채널 강화

11. 결론 및 검토 요청 사항

본 기획안은 직장인 특화 AI 영어 학습 시장의 블루오션을 선점하는 것을 목표로 하며,
6개월 내 시장 진입 후 1년 이내 손익분기점 도달을 계획한다.

검토 요청 사항:
- 5.6억 예산 중 마케팅 1.5억의 적정성 검토
- B2B 영업 인력 충원 시점 (Phase 3 vs Phase 4)
- LLM API 이중화 구조의 비용 효율성 재검토
