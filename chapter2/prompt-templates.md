# 프롬프트 템플릿 모음

실무에서 자주 사용하는 프롬프트 템플릿 3종을 정리한 문서입니다.
각 템플릿의 항목을 채워서 사용하면 일관되고 정확한 결과를 얻을 수 있습니다.

---

## 1. 비즈니스 이메일 작성

**템플릿 이름:** 이메일 작성 프롬프트

**사용 상황:** 미팅 요청, 협업 제안, 보고, 안내 등 업무용 이메일을 작성할 때 사용합니다. 수신자와의 관계, 톤, 분량을 지정하면 상황에 맞는 이메일을 빠르게 만들 수 있습니다.

**템플릿:**

```
역할: [작성자의 직책 또는 역할]
수신자: [받는 사람의 직책과 관계]
목적: [이 이메일로 달성하고자 하는 것]
참고 내용: [이메일 작성에 필요한 배경 정보]
형식:
  - 제목 포함
  - 분량: [원하는 분량]
  - 톤: [정중한 / 친근한 / 중립적]
조건: [금지 표현 또는 반드시 포함할 내용]
```

---

## 2. 데이터 시각화

**템플릿 이름:** 데이터 시각화 프롬프트

**사용 상황:** CSV, JSON 등의 데이터를 차트로 변환할 때 사용합니다. 차트 종류, 축 항목, 출력 형식을 지정하면 바로 시각화 코드를 생성하고 결과물을 저장할 수 있습니다.

**템플릿:**

```
역할: 데이터 시각화 전문가
분석 대상: [파일명 또는 데이터 내용]
요청: 위 데이터를 차트로 시각화해줘
형식:
  - 차트 종류: [막대 / 선 / 원형]
  - 제목: [차트 제목]
  - X축: [항목명], Y축: [항목명]
  - 결과물: PNG 파일로 저장
조건: 색상 구분 명확하게, 범례 반드시 포함
```

---

## 3. 데이터 분석

**템플릿 이름:** 데이터 분석 프롬프트

**사용 상황:** 수집된 데이터에서 의미 있는 패턴과 인사이트를 도출할 때 사용합니다. 분석 결과를 비전공자도 이해할 수 있는 형태로 정리해야 하는 보고서 작성, 의사결정 지원 등에 적합합니다.

**템플릿:**

```
역할: 데이터 분석 전문가
분석 대상: [분석할 파일명 또는 데이터 내용]
분석 목적: [이 분석을 통해 알고 싶은 것]
요청: 위 데이터를 분석해줘
형식:
  - 주요 수치 요약 (표 형식)
  - 주목할 만한 패턴 또는 이상값 설명
  - 인사이트 2~3가지
  - 개선 제안 1가지
조건: 전문 용어 없이 비전공자도 이해할 수 있게
```
