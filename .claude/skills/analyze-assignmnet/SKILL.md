# 매출 데이터 분석 자동화 Skill
## 입력
- 매출 CSV 파일 (주문번호, 주문일, 지역, 담당자, 제품분류, 제품명, 단가, 판매수량, 매출액, 영업이익, 이익률 등의 열 포함)
## 출력
- 정제된 CSV 파일 (cleaned.csv)
- 차트 이미지 3종 PNG (chart-bar.png, chart-line.png, chart-pie.png)
- 분석 보고서 (report.md)
## 실행 절차
### 1단계: 데이터 품질 진단
입력 파일을 읽어 다음 항목을 확인한다.
- 결측값 열별 개수
- 중복 행 개수
- 날짜 형식 통일 여부
진단 결과를 한 줄로 요약해서 출력한다.
### 2단계: 데이터 정제
다음 기준으로 자동 처리한다.
- 핵심 수치(판매수량, 매출액, 영업이익)가 모두 비어있는 행은 삭제
- 담당자가 비어있는 행은 '미확인'으로 채움
- 날짜 형식을 YYYY-MM으로 통일
- 중복 행 제거 (주문번호 기준)
처리 결과를 [원본파일명]-cleaned.csv로 저장한다.
### 3단계: 차트 3종 생성
정제된 파일을 기반으로 다음 차트를 순서대로 생성한다.
- 제품분류별 총 매출액 막대 차트 → chart-bar.png
- 월별 매출 추이 선 차트 → chart-line.png
- 지역별 매출 비중 원형 차트 → chart-pie.png
### 4단계: 보고서 작성 및 저장
분석 결과를 다음 구성으로 report_assignment.md 파일에 저장한다.
- 전체 요약
- 제품별 실적 표 + 서술 분석
- 지역별 실적 표 + 서술 분석
- 월별 추이 표 + 서술 분석
- 종합 의견 및 다음 분기 제언
### 5단계: 완료 보고
모든 단계가 완료되면 다음 형식으로 결과를 출력한다.
✅ 정제 완료: [원본 행 수] → [정제 후 행 수]
✅ 차트 생성: chart-bar.png / chart-line.png / chart-pie.png
✅ 보고서 저장: report_assignment.md
